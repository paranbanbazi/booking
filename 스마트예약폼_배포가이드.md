# 스마트 예약 폼 — 배포 가이드

> **구성**: Next.js 페이지 1개 + Supabase 테이블 1개
> **비용**: 무료 (Vercel Free + Supabase Free)
> **소요 시간**: 약 20~30분

---

## STEP 1. Supabase 테이블 만들기

Supabase 대시보드 → SQL Editor에서 아래 SQL을 실행하세요.

```sql
-- 상담 예약 테이블
CREATE TABLE bookings (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  consultation_type TEXT NOT NULL,    -- 상담 분야
  business_type TEXT NOT NULL,        -- 사업 형태
  name TEXT NOT NULL,                 -- 성함
  phone TEXT NOT NULL,                -- 연락처
  preferred_method TEXT NOT NULL,     -- 희망 상담 방법
  status TEXT DEFAULT 'new'           -- new / contacted / completed
);

-- 누구나 예약을 넣을 수 있도록 RLS 정책 설정
ALTER TABLE bookings ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Anyone can insert bookings"
  ON bookings FOR INSERT
  TO anon
  WITH CHECK (true);

-- 관리자만 조회 가능 (Supabase 대시보드에서 직접 확인)
CREATE POLICY "Only authenticated can view"
  ON bookings FOR SELECT
  TO authenticated
  USING (true);
```

## STEP 4. 배포

```bash
git add .
git commit -m "상담 예약 페이지 추가"
git push
```

Vercel에 연결되어 있으면 자동 배포됩니다.
배포 후 `도메인/booking` 으로 접속하면 예약 폼이 나옵니다.

---

## STEP 5. 링크인바이오 연결

리틀리(litt.ly) 또는 링크트리에서:
- 버튼 1: "상담 예약" → `https://도메인/booking`
- 버튼 2: "카카오톡 채널" → 카톡 채널 URL

인스타 프로필 웹사이트에 링크인바이오 URL 등록.

## STEP 6. 새 예약 확인 방법

### 방법 A: Supabase 대시보드에서 직접 확인
Supabase → Table Editor → bookings 테이블에서 새 예약을 확인할 수 있습니다.

### 방법 B: 이메일 알림 (선택사항, 나중에 추가 가능)
Supabase Database Webhooks 또는 Edge Functions로 새 예약 시 이메일 알림을 보낼 수 있습니다.
지금은 대시보드에서 직접 확인하는 것으로 충분합니다.
